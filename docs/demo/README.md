# Political Spectrum App - Demo Screenshots

This folder contains screenshots captured by the Playwright demo script.

## Running the Demo

```bash
# Install Playwright (if not already installed)
npm install -D @playwright/test
npx playwright install chromium

# Run the demo script
node playwright-demo.cjs

# Include AI analysis (requires configured API keys)
node playwright-demo.cjs --ai

# Record video
node playwright-demo.cjs --video
```

## Analysis Types

### Algorithm Analysis (Default)
- **No API key required**
- 3-layer scoring pipeline
- Outlet baseline + Article delta
- Evidence panel with reasoning
- Confidence scoring

### AI Analysis (Optional)
- Requires configured API keys
- Enhanced perspective summaries
- Detailed talking points
- Multi-provider support (ChatGPT, Claude, Gemini, etc.)

## Prerequisites

1. App running on `http://localhost:3000`
2. Playwright installed: `npm install -D @playwright/test`
3. Chromium browser: `npx playwright install chromium`
4. For AI demo: API keys configured in `.env`
