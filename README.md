## 🖥️ What this app does

political-spectrum-app-v2 helps you review news and media content for signs of political bias. It uses text analysis and AI support to show where a source may fall on the political spectrum. The app is built for everyday use, so you can open it, paste or load content, and get a clear result without needing technical skills.

Use it to:
- Check if an article leans left, center, or right
- Review sentiment in headlines and body text
- Compare different news sources
- See data in a simple visual format
- Track patterns in media coverage over time
